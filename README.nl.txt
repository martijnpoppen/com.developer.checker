# Krijg inzicht in jouw Homey Apps

Developer Checker checkt voor nieuwe installaties op jouw apps.
Laat bijvoorbeeld een lamp knipperen bij een nieuwe installatie.

Current features:
- Check app installatie verhoging
- Check app installatie verlaging
- Check app specifieke installaties
- Check totaal installaties van alle apps
